Typescript(이하 ts)의 특징 및 장점

  • React 등 사용영역이 점차 확대되고 있다.
  • 타입을 지원하므로 다양한 도구의 지원를 받기가 좋아 생산성이 향상된다.
  • Java 나 C# 등의 클래스 기반 언어를 먼저 알고 접근하면 더 좋다.
  • 컴파일러 기능을 내장하고 있으므로, Babel이 필요 없게 된다.
  • 코드 가독성, 예측성, 안정성이 향상되어 대규모 프로젝트에 적합하다.
