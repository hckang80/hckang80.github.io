자주 사용하는 단축키 정리(Vim, 터미널)

2018-05-26
etc
vim, 단축키, 터미널

1. Vim

h, j, k, l - 좌, 하, 상, 우 커서 이동

i: 현재 커서 위치에 Insert 하기

w: 단어 첫글자로 이동하기
b: 백워드 방향으로 단어의 첫글자로 이동하기

0 - 라인 맨 앞으로 이동
^ - 문장 맨 앞으로 이동
$ - 문장 맨 뒤로 이동

dd - 현재 줄 잘라내기
yy - 현재 줄 복사하기
p - 붙혀넣기
u - Undo
Ctrl + R : Redo

~ : 선택 문자 대소문자 변경

:w - 문서 저장하기
:q - 현재 문서 닫기
:q! - 저장하지 않고 닫기
:wq - 저장하고 닫기
:숫자 - 지정한 라인넘버로 이동
